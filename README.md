# Team-Manager

TaskPilot is a team task management web application built using Flask. The application helps teams create, manage, and track tasks efficiently through a clean dashboard interface.

## Features

* User Authentication (Login/Register)
* Task Creation and Assignment
* Task Status Tracking
* Dashboard Analytics
* Role-Based Access Control
* Priority and Due Date Management

## Tech Stack

* Flask
* SQLAlchemy
* SQLite
* HTML
* CSS
* JavaScript

## Deployment

The application is deployed on Railway.

Live URL:
https://web-production-3cb85.up.railway.app

## Installation

```bash
git clone https://github.com/kailashPatel1/team-task-manager.git
cd team-task-manager
pip install -r requirements.txt
python app.py
```

## Project Structure

* routes/ → Application routes
* templates/ → HTML templates
* static/ → CSS and JavaScript
* models.py → Database models
* services/ → Email and scheduler services
